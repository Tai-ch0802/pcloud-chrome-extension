# 專案概覽

HyperCmdC for pCloud 是一款強大的 Chrome 擴充功能，將 pCloud 整合至瀏覽體驗中。

## 描述
一款將 pCloud 整合至您的瀏覽體驗的強大 Chrome 擴充功能。只需點擊一下，即可將圖片、選取的文字和整個文件區段直接儲存至您的 pCloud 帳戶。

## 核心原則
本專案的架構設計旨在實現可擴展性、可維護性，並為未來的營利做好準備，清楚區分免費與付費功能。

## 關鍵功能
*   **右鍵選單整合**：在任何網頁上按右鍵即可存取 pCloud 工具。
    *   **儲存圖片**：將圖片直接上傳至 pCloud。
    *   **將選取範圍儲存為文字**：將選取的文字儲存為 Markdown 檔案。
    *   **將選取範圍儲存為文件**：將選取的內容（包含圖片）轉換為 Markdown (`.md`) 或 Word (`.doc`) 文件。
*   **智慧文件下載器**：
    *   自動下載並上傳選取內容中的圖片。
    *   更新 Markdown 中的圖片連結以指向已上傳的資產。
    *   支援「連線檢查」以確保下載可靠。
*   **可設定的檔名與路徑**：
    *   使用 `PAGE_TITLE` 和 `TIMESTAMP` 等變數自訂檔名模式。
    *   設定預設上傳資料夾。
    *   根據檔名路徑動態建立資料夾（例如 `Folder/Subfolder/Filename`）。
*   **安全驗證**：使用 pCloud 官方 OAuth 2.0 流程進行安全且無縫的登入。
*   **國際化**：完全支援英文與繁體中文在地化。
