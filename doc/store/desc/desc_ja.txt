🚀 HyperCmdC for pCloud：究極のクラウド活用ツール！【オープンソース＆無料】☁️

ブラウザから直接 pCloud の機能をフル活用！HyperCmdC は完全なオープンソースかつ無料となり、pCloud を日々のワークフローにシームレスに統合します。

✨ 主な機能：

🖼️ スマート画像保存
• 右クリックで任意の画像を pCloud に即座にアップロード。
• アップロードをカスタムフォルダに自動整理。
• 手動でのダウンロードや再アップロードはもう不要です！

📥 直感的なドラッグ＆ドロップ
• Webページから画像をオーバーレイに直接ドラッグしてアップロード！
• 手間なく高速です。

📝 テキスト選択を Markdown に
• テキストを選択 -> 右クリック -> 保存！
• クラウドに Markdown メモを即座に作成。
• リサーチやメモ取りに最適です。

📄 Web をドキュメント化
• 任意の Web コンテンツをクリーンな Markdown や Word ドキュメントに変換。
• 画像は自動的にダウンロードされ、pCloud にアップロードされ、ドキュメントにリンクされます！🤯
• リッチで持ち運び可能なメモを保持します。

📂 スマート PDF 統合
• Chrome で PDF を表示すると、美しいフローティングボタンが表示されます。
• ワンクリックで PDF を特定の pCloud フォルダに直接保存。
• AI や他のアプリ用に PDF を Markdown に変換することも可能！

⚡ パワーユーザー向けツール
• カスタマイズ可能なファイル名（日付、タイトル、カスタムキー）。
• ドメインルール：Web サイトごとに異なるアップロードフォルダを設定！
• スマートフォールバックにより、ファイルを失うことはありません。

🔒 セキュリティとプライバシー
• オープンソースコードベース - GitHub で確認可能！
• 公式 pCloud OAuth 2.0 を使用。
• 認証情報がブラウザから外部に出ることはありません。

HyperCmdC for pCloud で今すぐ生産性を向上させましょう！🚀
