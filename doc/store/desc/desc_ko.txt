🚀 HyperCmdC for pCloud: 최고의 클라우드 절약 도구! [오픈 소스 및 무료] ☁️

브라우저에서 직접 pCloud의 모든 기능을 활용하세요! HyperCmdC는 이제 완전한 오픈 소스이자 무료이며, pCloud를 일상적인 워크플로에 원활하게 통합합니다.

✨ 주요 기능:

🖼️ 스마트 이미지 저장
• 마우스 오른쪽 버튼을 클릭하여 이미지를 pCloud에 즉시 업로드하세요.
• 업로드를 사용자 지정 폴더로 자동 정리합니다.
• 더 이상 수동으로 다운로드하고 다시 업로드할 필요가 없습니다!

📥 직관적인 드래그 앤 드롭
• 웹 페이지의 이미지를 오버레이로 직접 드래그하여 업로드하세요!
• 간편하고 빠릅니다.

📝 텍스트 선택을 Markdown으로
• 텍스트 강조 표시 -> 마우스 오른쪽 버튼 클릭 -> 저장!
• 클라우드에 Markdown노트를 즉시 생성합니다.
• 조사 및 메모 작성에 완벽합니다.

📄 웹을 문서로
• 모든 웹 콘텐츠를 깔끔한 Markdown 또는 Word 문서로 변환하세요.
• 이미지가 자동으로 다운로드되어 pCloud에 업로드되고 문서에 연결됩니다! 🤯
• 메모를 풍부하고 휴대하기 좋게 유지하세요.

📂 스마트 PDF 통합
• Chrome에서 PDF를 볼 때 아름다운 플로팅 버튼이 나타납니다.
• 한 번의 클릭으로 PDF를 특정 pCloud 폴더에 직접 저장하세요.
• AI 또는 다른 앱을 위해 PDF를 Markdown으로 변환을 시도합니다!

⚡ 고급 사용자 도구
• 사용자 지정 가능한 파일 이름 (날짜, 제목, 사용자 지정 키).
• 도메인 규칙: 웹사이트마다 다른 업로드 폴더를 설정하세요!
• 스마트 폴백으로 파일을 잃어버리지 않도록 보장합니다.

🔒 보안 및 개인 정보 보호
• 오픈 소스 코드베이스 - GitHub에서 직접 확인하세요!
• 공식 pCloud OAuth 2.0을 사용합니다.
• 자격 증명은 브라우저를 벗어나지 않습니다.
